Общее описание

Сервис позволяет получить информацию о том, сколько раз заданная строка keyword встречается на веб-странице по адресу url. Информация выводится отдельно для заголовка и для тела страницы, т.е. поиск ведётся отдельно в <head> и <body>.

 

Входные данные

Входные данные передаются по методу GET либо POST.

 

Имя параметра

Обязательный?

Описание

url

Да

URL страницы, где будет происходить поиск. Допустимые протоколы – http и https. Указание протокола в начале url необязательно.

keyword

Да

Искомая строка. Может содержать любые символы.

case_sensitive

Нет

Если case_sensitive=true, поиск будет выполняться с учётом регистра

 

Пример запроса к сервису: <адрес веб-сервиса>?url=vk.com&keyword=vk&case_sensitive=true

 

Выходные данные

Выводятся в формате JSON со следующими ключами:

  • success  – равен true, если страница была успешно загружена, иначе false.
  • message – возвращается, если success=false. Содержит сообщение об ошибке.
  • matches  – возвращается, если success=true. Содержит объект с ключами head и body. Эти ключи содержат количество вхождений строки keyword в заголовок/тело страницы соответственно и -1, если соответствующий тег не найден.

 

Пример выходного документа при успешной загрузке страницы:

﻿{"success":true,"matches":{"head":2,"body":24}}.

 

Примеры выходного документа при ошибке загрузки страницы:

{"success":false,"message":"URL is invalid"}.

{"success":false,"message":"Missing parameters: keyword"}

{"success":false,"message":"Could not resolve host: vkddd.com"}

 

Пример клиентской HTML-страницы

<html>

<head></head>

<body>

<form action="http://localhost/test/old" method="get">

               Keyword:                       <input type="text"         name="keyword"><br/>

               URL:               <input type="text"         name="url"><br/>

               Case sensitive <input type="checkbox"               name="case_sensitive" value="true" >

               <input type="submit">

               </form>

</body> 

</html>

 

Пример клиентского приложения на РНР

<?php

header('Content-Type: text/html; charset=utf-8');

 

if (isset($_GET['run'])){

 

//Считываем переданные параметры

$url=$_GET['url'];

$keyword=$_GET['keyword'];

$case_sensitive=isset($_GET['case_sensitive']) ? 'checked': '';

 

                    //----------Запрос к веб-сервису. BEGIN-------------

 

                    //Для правильной работы этот файл должен лежать в одной папке с index.php

                    $service_url='http://'.dirname($_SERVER['SERVER_NAME'].$_SERVER['REQUEST_URI']).'/';

                    $params=array('keyword'=>$keyword, 'url'=>$url);

                    if ($case_sensitive) $params['case_sensitive']='true';

                   

                    $context = stream_context_create(array(

                                         'http' => array(

                                                             'method' => 'POST',

                                                             'header' => array('Accept: application/json',

                                                             'Content-Type: application/x-www-form-urlencoded'),

                                                             'content' => http_build_query($params)

                                         )

                    ));

                    $result = 'Result is:<br />'.file_get_contents($service_url, false, $context);

                    //-----------Запрос к веб-сервису. END--------------

}

//Иначе устанавливаем их по умолчанию

else {

                    $url="http://example.com";

                    $keyword='example';

                    $case_sensitive='';

                    $result='';

}

 

print

'<html>

 <head><style>input{margin:2;}</style></head>

 <body>

                    <form action="'.$_SERVER['PHP_SELF'].'" method="get">

                                         Keyword: <input type="text" name="keyword" value="'.$keyword.'"><br/>

                                         URL:<input type="text" size="70" name="url" value="'.$url.'"><br/>

                                         <input type="checkbox" name="case_sensitive" value="1"  '.$case_sensitive.'>Case sensitive <br />

                                         <input type="submit" name="run" value="Run!">

                    </form>

                    <br/>

                    '.$result.'

 </body>   

</html>';

 

?>
